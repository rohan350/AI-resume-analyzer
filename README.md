# AI Resume Analyzer & Job Matcher

A full stack AI powered web application that analyzes resumes and extracts skills.

## Features
- Resume skill extraction
- Job matching
- Resume improvement suggestions
- Analytics dashboard

## Tech Stack

Frontend
- React.js
- Chart.js

Backend
- Node.js
- Express.js

AI Service
- Python
- Flask
- NLP

Database
- MySQL / MongoDB

## Setup

Backend
cd backend
npm install
node server.js

AI Service
cd ai_service
pip install flask pandas nltk
python app.py

Frontend
cd frontend
npm install
npm start
